## TIL 작성 목적 

- 안드로이드 최신 기술에 꾸준한 관심을 갖고 공부하되, 정확한 근거를 갖고 프로젝트에 적용하기 위해서 
- 평소에 개발하면서 새롭게 알게 된 점이나 깊이 있게 공부한 내용을 기록하기 위해서
- 기술 면접 때 이력서에 작성한 기술 스택에 대해 자신감 있게 설명할 수 있도록 

## 커밋 메시지 컨벤션 

- `add` : 파일 추가 
- `mod` : 파일 수정
- `del` : 파일 삭제
- `chore` : 사소한 수정 작업 (오타, 코드 정렬 등)
- `move` : 파일 위치 이동 
- `rename` : 파일 또는 폴더 이름 수정
